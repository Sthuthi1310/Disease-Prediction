# Disease Prediction App

## Overview
A machine learning-based web application that predicts diseases based on user symptoms.

## Features
- Patient details input
- Symptom-based prediction
- Top 3 disease prediction
- Confidence score
- Severity analysis
- Precautions and description

## Tech Stack
- Python
- Streamlit
- scikit-learn
- pandas

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
